# **App Name**: Editors Table Timesheet

## Core Features:

- User Authentication: User login/authentication with roles (admin, editor).
- Time Record Creation: Editor time record creation: Date, project name, project type, project duration, completion timestamp, revision flag. No database required for MVP, data is stored client-side only. Editors can edit there timesheet.  
Editors can mark as complete
- Editor Report: Editor's personal time report view with date range selection.
- User Management: Admin ability to manage users (creation/deletion is mocked in the UI).
- Admin Time Report: Admin to select date ranges, and view a mocked chart visualizing editor's time entries. Project types can be defined in code. once complete the task admin should notify it as chrome notification in realtime

## Style Guidelines:

- Primary color: Deep blue (`#F00000`) to convey professionalism.
- Secondary color: Light gray (#E0E0E0) for backgrounds and neutral elements.
- Accent: Teal (#1C1C1C) for interactive elements and highlights.
- Clean, modern, and readable fonts such as Open Sans for UI.
- Use a set of professional and consistent icons from Material Design Icons.
- A clean, intuitive, and responsive layout using a grid system.
- Use primerect
